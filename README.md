# 📊 Student Performance Prediction System

## 📌 Project Overview
This project is a web-based application that predicts student performance using Machine Learning.  
It analyzes student scores and provides insights through visualizations.

The system is built using Python, Flask, and basic data analysis techniques.

---

## 🚀 Features
- 📈 Predict student performance using ML model
- 📊 Visualize subject-wise scores (Math, Reading, Writing)
- 🧠 AI-based prediction comparison
- 🌐 Simple web interface using Flask
- 📂 Cleaned dataset for analysis

---

## 🛠️ Technologies Used
- Python
- Flask
- Pandas
- Matplotlib
- HTML, CSS
- Machine Learning
